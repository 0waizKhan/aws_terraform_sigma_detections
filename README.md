# AWS TerraForm Sigma Rules
Sigma rules for detecting suspicious activity across AWS environments. Includes detections for CloudTrail, IAM, VPC, Lambda, and GuardDuty. Designed to support threat hunting, monitoring, and incident response in cloud security operations.

## Features

- AWS-focused Sigma detection rules  
- Mapped to MITRE ATT&CK techniques  
- Coverage across AWS services:
  - CloudTrail
  - IAM activity
  - GuardDuty alerts
  - VPC network behavior
  - Lambda execution anomalies
